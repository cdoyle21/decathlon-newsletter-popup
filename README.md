# decathlon-newsletter-popup

A newsletter popup created using HTML, JS and CSS for Decathlon

Implements a static express server and fetches that data on the front end.

## How to Install and run

1. `git clone git@github.com:cdoyle21/decathlon-newsletter-popup.git`
2. `npm i`
3. `npm run start-server`
4. Open newsletter-popup/index.html in browser
